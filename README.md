## 🚀 Latest Project / 最近项目

### 📦 Asset QR Inventory System / 资产扫码管理系统

A mobile-first asset tracking system for IT operations and real-world device management.

一个面向 IT 运维场景的移动资产管理系统。

---

### ✨ Key Features / 核心功能

- 📷 QR Code & Barcode scanning (dual-mode support)
- ✍️ Inline editing (edit-in-place, no page reload)
- 📱 Mobile-first responsive UI
- 💾 LocalStorage persistence (offline usable)
- 📤 Excel export (SheetJS)
- 🖼 OCR recognition (Tesseract.js)
- 🔍 Asset lifecycle tracking (scan → record → export)

---

### 🧠 Tech Stack / 技术栈

- HTML5 + CSS3 + JavaScript (Vanilla)
- html5-qrcode (camera scanning)
- Tesseract.js (OCR recognition)
- SheetJS (Excel export)
- Deployment: GitHub Pages / Ubuntu Caddy / Nginx
